🔍 Reality Check: What Actually Works

✅ What's Working:

  • • Claude Code's built-in agent types (general-purpose, team-lead, etc.)
  • • These work through Claude's internal Task tool
  • • They have full tool access as configured
  • • PROACTIVE agents can self-activate

❌ What's Not Working:

  • • Custom agents in .claude/agents/ cannot be invoked
  • • No Task tool available for custom agents
  • • No inter-agent communication
  • • No shared memory or state management

📝 How to Actually Use Sub-Agents Today

# This WORKS with Claude's built-in agents: Task( subagent_type="general-purpose", # Built-in agent type prompt="search for implementation details", description="find sub-agent info" ) # This DOESN'T WORK with custom agents: Task( subagent_type="hub-keeper", # Custom agent from .claude/agents/ prompt="coordinate hub activities", description="manage hub" ) # ERROR: Agent type 'hub-keeper' not found
